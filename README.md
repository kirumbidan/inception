## Support Ticket Analytics & Prediction

### [Try out the model](https://huggingface.co/spaces/Kirumbi/support_tickets_model)

Objective  
Use the [Kaggle Customer Support Ticket Dataset](https://www.kaggle.com/datasets/suraj520/customer-support-ticket-dataset) to:

* Analyze and visualize:  
  * Distribution by ticket types  
  * Distribution by ticket priority  
  * Distribution by support channel  
  * Customer Demographics (age, gender)  
* Identify:   
  * Most common issues  
  * Most used support channels  
  * Products with highest ticket volume  
* (BONUS) Predict ticket type using:   
  * Ticket Subject  
  * Ticket Description

  Here the Approach to use is:

* Text preprocessing using NLTK or spaCy  
* Feature Extraction using scikit-learn  
* Use a logistic regression model for classification  
* Separate the dataset into 2 for training & testing  
* Evaluate performance

Library Constraints:

* NLTK(Natural Language ToolKit)  
* spaCy  
* Scikit-learn  
* Pandas  
* NumPy  
* Any other open source python library you may need \- do NOT use LLMs / paid API SAAS integrations
